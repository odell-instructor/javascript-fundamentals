### Running Code
- Using Code Runner, you can run your JavaScript file in multiple ways
    - Right-Click on the file and choose "Run Code"
    - A triangle play button on the upper right side of VS Code
    - Keyboard shortcut Ctrl + alt + n
- Can run the file from the Termain using the following command
    - node main.js
    - The terminal cursor needs to be in the same folder as the file, for the command to work properly.
