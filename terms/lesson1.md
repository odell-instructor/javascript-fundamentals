# Lesson 1 - Introduction to JavaScript

### Code Comments
- // for a single comment
- /* comment */ for block or multi-line comment
- Not executed when the file is ran
- keyboard shortcut: Ctrl + / on Windows
- keyboard shortcut: Command + / on Mac

### Code Standards
- Clean and concise code
- Organized

### Naming
Naming should have:
- A Purpose: What it is for and how it behaves
- A Concept: How to think about it, or an idea
- A Contract: How it works

### Variables (Identifiers)
- Store values as nouns
- Begin with a letter, underscore, or dollar sign
- Case Sensitive

### Variable Declarations
- const: Declares a block or scope
- var: Declares a function scope or global scope variable
- let: Declarea a variable in a block or expression

### Naming and Variable example
- cameCase style
- all caps for **const** conventionally
- *Undefined* for variables without data

`let variableName = 'Hello';`
